# 任务清单列表

*  1.每天合理安排自己的时间，列出自己每天需要做的工作？
*  2.我们可以使用吞蛤蟆的概念去完成我们的任务列表。
*  3.什么是吞蛤蟆的概念？
*  4.我们需要将每天的工作一一列表出来，挑选其中的**_“大蛤蟆”_**（比较难处理的任务），在我们早上来到办公室的时候是每天早上最清醒的时候，我们把这个“大蛤蟆”吞掉之后会让我们整个一天都有成就感，并且剩下的时间只需要处理一些简单的问题。
